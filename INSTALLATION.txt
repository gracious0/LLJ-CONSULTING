PROBLEM: Links don't work
SOLUTION: Ensure all .html files are in the same folder

PROBLEM: Styling looks wrong
SOLUTION: 
  1. Make sure styles.css is in assets/css/
  2. Clear browser cache (Ctrl+Shift+Delete)
  3. Refresh page (Ctrl+R)

PROBLEM: Mobile menu doesn't work
SOLUTION:
  1. Make sure script.js is in assets/js/
  2. JavaScript might be disabled - check browser settings
  3. Try a different browser

PROBLEM: Contact form doesn't send emails
SOLUTION:
  1. Set up email integration (see Step 6 above)
  2. Test on actual website (not local file)
  3. Check spam folder for test emails
