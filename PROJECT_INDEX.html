üîß Customization Guide

Change Company Information

Phone: Search for "(505) 555-1234" in all HTML files
Email: Search for "info@lljconsulting.com" in all HTML files
Address: Search for "500 4th St NW, Suite 102, Albuquerque, NM 87102" in all HTML files

Change Colors

Edit assets/css/styles.css - Update the CSS variables in the `:root` section (lines 7-18)

Update Content

  • Home: Edit index.html - Hero section, testimonials, services
  • About: Edit about.html - Company description, benefits
  • Services: Edit services.html - Service details and descriptions
  • Contact: Edit contact.html - Contact information

Set Up Email

The contact form currently displays a success message but doesn't send emails.

Option 1 - Formspree: Go to formspree.io, create a form, get the form ID, update the form action in contact.html
Option 2 - Netlify: Deploy the site on Netlify, forms work automatically
Option 3 - Custom: Update script.js to post to your backend API
